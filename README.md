# 🕌 Htmlazan — Application de Prières Islamique

Une application web complète dédiée aux **prières islamiques**, développée en **HTML/CSS/JavaScript** pur et déployée sur Vercel.

## À propos

**Htmlazan** est une Progressive Web App (PWA) islamique offrant tous les outils essentiels pour le pratiquant musulman : horaires de prière, direction de la Qibla, calendrier islamique, compteur de dhikr et bien plus encore.

## 🌐 Voir l'application en ligne

👉 [Ouvrir Htmlazan](https://swayeb1.vercel.app/)

## ✨ Fonctionnalités

- 🕐 **Horaires de prière** — affichage des 5 prières quotidiennes selon la localisation
- 📅 **Calendrier islamique** — dates hijri et événements religieux
- 🧭 **Direction Qibla** — boussole pour trouver la direction de La Mecque
- 🕌 **Mosquées à proximité** — localisation des mosquées proches
- 📿 **Tasabih** — compteur de dhikr et invocations
- 🔊 **Audio Adhan** — appel à la prière en audio
- ⚙️ **Paramètres** — personnalisation de l'application
- 📖 **À propos** — informations sur l'application
- 🚀 **Onboarding** — guide de démarrage pour les nouveaux utilisateurs

## 📁 Structure du projet

```
htmlazan/
├── index.html          # Page principale (horaires de prière)
├── about.html          # Page à propos
├── adjust.html         # Ajustement des horaires
├── calendar.html       # Calendrier islamique
├── mosques.html        # Mosquées à proximité
├── onboarding.html     # Guide de démarrage
├── qibla.html          # Direction de la Qibla
├── settings.html       # Paramètres
├── tasabih.html        # Compteur de dhikr
├── audio/              # Fichiers audio (Adhan)
└── .snapshots/         # Snapshots de l'application
```
